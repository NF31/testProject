🎮 Interface de démarrage – Sélection personnalisée du jeu

Avant de commencer une partie, le joueur accède à une interface de démarrage interactive, inspirée des jeux classiques comme Super Mario Bros..

Cette interface permet de personnaliser l’expérience de jeu en choisissant plusieurs paramètres.

👤 Sélection des joueurs

Le joueur peut choisir entre plusieurs versions de la chocolatine :

🍫 Chocolatine classique

🤍 Chocolatine chocolat blanc

🥇 Chocolatine dorée (débloquée après un certain score)

Chaque personnage possède :

Un design différent

Une animation spécifique

Une petite capacité spéciale (ex : saut légèrement plus haut)

Cela permet de rendre le jeu plus personnalisable et rejouable.

🎚️ Choix du niveau de difficulté

Le joueur peut sélectionner la difficulté :

🟢 Facile

Obstacles lents

Moins d’obstacles volants

Pas de double obstacles

🟡 Moyen

Vitesse normale

Apparition d’obstacles volants

Quelques doubles obstacles

🔴 Difficile

Vitesse rapide

Obstacles fréquents

Double obstacles

Apparition régulière du Beluga (référence à Airbus)

La difficulté modifie :

La vitesse du jeu

La fréquence des obstacles

Le multiplicateur de score

🗺️ Choix des niveaux (Carte interactive)

L’interface propose une carte de progression, avec plusieurs niveaux à débloquer.

Niveau 1 – Centre-ville

Place du Capitole

Quai de la Daurade

Niveau 2 – Sport

Stadium de Toulouse

Stade Toulousain

Niveau 3 – Industrie & Futur

Airbus

Cité de l’Espace

Le joueur doit terminer un niveau (atteindre le drapeau final) pour débloquer le suivant.

🎮 Différents modes de jeu

Nous avons également imaginé plusieurs modes :

🏃 Mode Classique → progression normale

♾️ Mode Endless → course infinie avec score maximum

⏱️ Mode Chrono → survivre un temps limité

⭐ Mode Bonus → collecte d’étoiles uniquement

🆚 Mode versus → 2 joueurs s’affrontent 

Cela permet de varier l’expérience et d’augmenter la rejouabilité.

🌍 Choix des villes (Extension possible)

Bien que le jeu principal se déroule à Toulouse, une extension permettrait de choisir d’autres villes.

Chaque ville proposerait :

Des monuments différents

Un décor spécifique

Des obstacles adaptés

Cela rend le projet évolutif et extensible.
